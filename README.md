# Order Management System (C++ Console App)

A simple command-line Order Management System that simulates an e-commerce backend with product listings, dynamic inventory tracking, cart management, order placement, and simulated shipping.

## Features

- **Product Catalog:** View available products with live stock.
- **Add to Cart:** Select products and quantity to build your shopping cart.
- **View Cart:** See and review current items in your cart.
- **Order Placement:** Place orders if your cart has items and stock is available.
- **Shipping Simulation:** Orders are 'shipped' after a brief delay, status updates shown.
- **Inventory Management:** Product stock updates automatically with every order.

## Technologies Used

- **Language:** C++
- **Standard Library:** STL containers (`unordered_map`, `vector`), strings, classes, threads, IO.

## How To Run

1. **Clone or Download This Repository**
2. **Open a terminal** and navigate to the project directory.
3. **Compile the code:**
    ```
    g++ -std=c++11 -o order-management main.cpp
    ```
4. **Run the program:**
    ```
    ./order-management
    ```
5. **Follow the menu instructions** in the terminal to interact (view products, add to cart, place orders, etc).
